type word = AST.letter list

Incremental Interface

type configuration
val initial : AST.pda -> configuration
val step_letter : AST.letter -> configuration -> configuration
val steps_word : word -> configuration -> configuration

Loops step_letter on a whole word.

val accepting : configuration -> bool

Checks whether the given configuration is accepting.

Monolithic Interface

val accepts : AST.pda -> word -> bool

Checks whether the given automaton accepts the given word. accepts pda word is simply pda |> initial |> steps_word word |> accepting.
